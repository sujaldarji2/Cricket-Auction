# 🏏 Cricket Auction System (DPPL)

A complete web-based Cricket Auction Management System developed for **DPPL (Dharti Pooja Premier League)**. This platform digitizes the traditional cricket auction process with real-time bidding, team management, and an interactive user interface.

---

## 🌟 Key Highlights
- 🔴 Real-time live auction system
- 💰 Automatic purse calculation & updates
- 👥 Dynamic team squad building
- 🎯 Player categorization (Base price system)
- 📊 Auction insights & bid tracking
- 🎲 Random player selection mode
- ⚡ Fast and responsive UI for smooth auction flow

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL

## 🔐 User Roles
- 👑 Admin: Controls auction, selects players, manages teams  
- 🧑‍🤝‍🧑 Teams: Participate in bidding and build squads  

---

## ⚙️ How It Works
1. Admin starts the auction  
2. Player appears randomly / manually  
3. Teams place bids in real-time  
4. Highest bidder wins the player  
5. Player gets added to the team squad  
6. Purse amount updates automatically
